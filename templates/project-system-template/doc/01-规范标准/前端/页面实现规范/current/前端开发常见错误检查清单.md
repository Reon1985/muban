# 前端开发常见错误检查清单

## 1. 使用时机

在页面开发、联调、自测、回归前后，统一按本清单复核。

## 2. 默认检查项

1. 页面按钮文案是否误显示 i18n key。
2. 搜索组件样式、宽度、占位文案、交互方式是否和规范一致。
3. 远程选择组件是否误加载全量数据，导致性能问题。
4. 列表页是否出现重复按钮、重复列、重复渲染。
5. 详情页是否遗漏只读规则、删除校验、引用校验。
6. 日期类字段在业务页面是否误显示到时分秒。
7. 外键引用是否统一只存主数据代码，并通过原表回显名称。

## 3. 回填原则

1. 新出现的高频问题，必须补充到本清单。
2. 一次性问题不要在这里重复堆积，优先沉淀可复用检查口径。
