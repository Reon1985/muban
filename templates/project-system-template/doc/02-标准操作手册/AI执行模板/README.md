## 2. 使用原则

1. 本目录承接“可以直接套用”的 AI 执行模板。
2. 长期原则写在 `01-规范标准/`，模板和执行骨架写在 `02-标准操作手册/`。
3. 使用模板时可以裁剪，但不应删除关键门禁和回填项。
4. 如果任务来源于测试报告驱动的问题修复，优先使用“测试报告问题批量收口实施模板”，不要直接从空白实施文档起草。
