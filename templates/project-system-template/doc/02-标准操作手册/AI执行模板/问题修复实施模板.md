# 问题修复实施模板

## 1. 使用说明

本模板用于承接“非测试报告驱动”的通用问题修复实施文档。

适用场景：

1. 单点 Bug 修复
2. 线上或联调发现的问题修复
3. 排障后需要形成正式修复计划的任务

不适用场景：

1. 来源于测试报告、QA 缺陷清单，且需要统一发版与统一验证
2. 已经扩大为新的正式版本需求包

如果属于测试报告驱动修复，优先改用：

- [测试报告问题批量收口实施模板.md](__PROJECT_ROOT__/doc/02-标准操作手册/AI执行模板/测试报告问题批量收口实施模板.md)

相关规则：

- [问题修复文档归档与执行规范.md](__PROJECT_ROOT__/doc/01-规范标准/文档治理/AI阅读路径规范/current/问题修复文档归档与执行规范.md)
- [测试报告驱动问题批量收口规则.md](__PROJECT_ROOT__/doc/01-规范标准/文档治理/AI阅读路径规范/current/测试报告驱动问题批量收口规则.md)
