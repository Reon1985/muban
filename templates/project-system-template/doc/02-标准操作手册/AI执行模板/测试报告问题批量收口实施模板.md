# 测试报告问题批量收口实施模板

## 0. 使用说明

本模板只用于“测试报告 / QA 缺陷清单 / 联调报告驱动”的整轮问题收口。

适用场景：

1. 问题来源已成批冻结
2. 本轮需要统一发版
3. 本轮需要统一验证

不适用场景：

1. 单点 Bug 修复
2. 线上反馈的临时修复
3. 排障后形成的通用修复任务，但不需要整轮批量收口

如果不满足以上条件，改用：

- [问题修复实施模板.md](__PROJECT_ROOT__/doc/02-标准操作手册/AI执行模板/问题修复实施模板.md)
