# 项目导航

## 触发条件

当用户或 AI 需要快速定位项目文档或代码时触发。

用于帮助快速找到"应该先看什么"。

## 核心原则

**先按任务定位文档，再按文档定位代码，不从正文和代码开始盲扫。**

## 最快阅读路径

### 新任务进入

```text
CLAUDE.md
→ 项目文档总览
→ 对应目录 README
→ 正文
```

### 进入开发前

```text
CLAUDE.md
→ 项目文档总览
→ 当前生效版本说明
→ 对应版本包 README
→ 当前已确认实施文档
→ 开发规范
→ 当前模块需求 / 页面规则 / 技术方案正文
```

### 快速定位代码前

```text
项目文档总览
→ 前端.md / 后端.md / 权限架构说明
→ 目标模块
→ 目标文件
→ 文档不足时再扩大搜索
```

## 注意事项

- 先走索引，不默认全量扫正文和代码
- 文档已有结论，优先复用
- 只有当文档无法支撑判断时，才扩大到全库搜索
